# PRD: Fugue — クロスメディア創作キュレーションプラットフォーム

**日付**: 2026-04-03
**製品段階**: 新規（サイドプロジェクト / ポートフォリオ）
**マスコット**: ヘッドセットをつけて筆を持ったフグ（Fugue ≈ Fugu）

---

## 1. 概要

Fugueは、クロスメディア創作キュレーションプラットフォームである。Pinterestが画像に限定されるのに対し、Fugueは音楽、イラスト、映像、文章、コード、3D、ゲームなど、あらゆる分野の創作物をジャンルを越えて一つのフィードで発見・キュレーションできる。

- 外部創作物のURLを「ピン」→ OGメタデータで自動プレビュー生成
- ピンをボードにまとめてテーマ別コレクションを構成
- クリック・ピン行動に基づいて好みを学習し、パーソナライズされたフィードを提供
- 作品詳細ページで関連作品を自動推薦

## 3. 背景

### 3.1 コンテキスト

創作物はプラットフォームごとに散在している。音楽はSoundCloud、イラストはpixiv、映像はYouTube、コードはGitHub。各プラットフォームは自分の分野のみを扱うため、「この音楽に合うイラスト」や「このゲームに合うBGM」を発見する方法がない。

既存のキュレーションサービス（Pinterest、Are.naなど）は主に画像に特化しており、音楽やコードなどのメディアを同等に扱えない。

### 3.2 なぜ今か？

- インディー/趣味の創作市場が持続的に成長中（VTuber、インディーゲーム、同人音楽など）
- 創作物が散在するプラットフォームが増え続けている
- Pinterestは画像中心。Are.naはニッチ。クロスメディアキュレーションは空白地帯

### 3.3 リファレンス

- **Pinterest**: 画像キュレーションの教科書。ピン＋ボード＋推薦フィードモデル
- **Are.na**: リサーチ/アーカイブ目的のキュレーション
- **Piapro**: 創作物投稿＋ライセンス共有。ボーカロイドエコシステム限定
- **Raindrop.io**: ブックマークマネージャー。キュレーションより整理ツール

## 6. ソリューション

### Feature 1: ピン (Pin)
外部創作物URLのキュレーション。OG自動プレビュー。ドメインベースの分野自動検出。

### Feature 2: ボード (Board)
ピンをテーマ別にまとめるコレクション。公開/非公開設定。

### Feature 3: 推薦フィード
ユーザーの好みに基づく作品推薦。v1: タグ頻度ヒューリスティック。ロードマップ: ヒューリスティック → フィーチャーストア → ML。

### Feature 4: 暗黙的嗜好学習
view、pin、board_addイベントをinteractionsテーブルに記録。

### Feature 5: 関連作品
作品詳細ページ下部にタグ類似度ベースの関連作品を表示。最大10件。

### Feature 6: 認証（実装済み）
Google OAuth、Discord OAuth。JWTベース認証。
