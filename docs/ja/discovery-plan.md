---

## 1. 背景

### 個人的な課題

自作曲のYouTube MVを作りたいが、イラスト・映像編集スキルがない。
同じ悩みを持つインディークリエイターは多いはず。

### 参考サービス

- **piapro** (ピアプロ): ボカロ文化圏で創作物を共有・コラボするプラットフォーム。ニコニコ動画エコシステムとの連携が強み。

### 現状の課題

- Twitter DMで非構造的な1:1マッチングが行われている
- 「イラストレーター募集します」→ ポートフォリオ確認 → DMやり取り → 条件交渉
- プロセスが非効率で、作品の相性よりも人脈に依存している

---

## 2. ターゲットユーザー

> 収益よりコラボ機会を求める趣味/インディークリエイター

### ペルソナ

| ペルソナ | 説明 | ニーズ |
|----------|------|--------|
| インディー作曲家 | 自作曲はあるがMV制作スキルがない | 自分の音楽に合うイラスト・映像を探したい |
| 趣味イラストレーター | 描いた絵を活用してもらいたい | 自分の絵に合う音楽プロジェクトに参加したい |
| 映像編集者 | 素材があればMV制作可能 | 音楽+イラスト素材を探してMVを作りたい |

### ライセンスモデル

デフォルトライセンス: **「クレジット表記で自由利用」**

- 商用利用不可（個人・趣味プロジェクト限定）
- クリエイター名のクレジット表記必須
- 二次創作・コラボ利用OK
- クリエイターがいつでもライセンスを変更可能

## 3. 市場調査

### ポジショニングマップ

```
        有料
         │
  Skeb   │   VGen
  ココナラ │   Fiverr
         │
─────────┼─────────
         │
  BOOTH  │   Fugue ★
  nizima │   (ここが空白地帯)
         │
        無料

  1:1依頼        作品マッチング
```

### 競合分析

| サービス | 種別 | 価格 | マッチング方式 | Fugueとの差別化 |
|----------|------|------|----------------|-----------------|
| **Piapro** | 創作物共有 | 無料 | 手動検索 | ボカロ特化、レコメンドなし |
| **VGen** | コミッション | 有料 | 1:1依頼 | 人ベース、有料のみ |
| **Skeb** | コミッション | 有料 | 1:1依頼 | リクエスト型、コラボではない |
| **ココナラ** | スキルマーケット | 有料 | 1:1依頼 | 汎用マーケット、作品マッチングではない |
| **Fiverr** | フリーランスマーケット | 有料 | 1:1依頼 | 海外向け、高価格帯 |
| **Ko-fi** | クリエイター支援 | 寄付制 | なし | 資金調達中心、マッチングなし |
| **ArtStation/Behance** | ポートフォリオ | 無料 | なし | 展示中心、コラボ機能なし |
| **BOOTH** | 素材販売 | 有料 | 手動検索 | 販売型、コラボではない |
| **nizima** | Live2D素材 | 有料 | 手動検索 | Live2D特化、コラボではない |

### 重要なインサイト

> **「無料 + 作品ベースのコラボマッチング」は誰もやっていない。**

- 有料コミッションプラットフォームは多いが、趣味クリエイター向けの無料コラボマッチングは空白地帯
- piaproが最も近いが、レコメンド機能がなく、ボカロ文化圏に限定されている
- Fugueはこの空白地帯を狙う

## 4. コアコンセプト

### 人ではなく「作品」をマッチングする

従来のアプローチ:
> 「イラストレーター募集します」→ ポートフォリオ確認 → 人を選ぶ

Fugueのアプローチ:
> 「自分の音楽に合うイラストを探したい」→ 作品同士をマッチング → コラボ提案

### なぜ作品ベースなのか？

1. **趣味クリエイターはポートフォリオを持たない** — 個別の作品は持っている
2. **スキルレベルより作品の相性が重要** — 趣味コラボでは「上手さ」より「雰囲気の一致」
3. **心理的ハードルが低い** — 人に声をかけるより、気に入った作品に反応する方が自然
4. **クロスドメイン発見** — 音楽を聴いて「この曲に合う絵」を見つける体験

---

## 5. コアフロー

### Step 1: 投稿 (30秒)

```
リンク貼り付け → 自動プレビュー → タグ選択 → 投稿完了
```

- YouTube、SoundCloud、Twitter、Pixiv等の外部リンクを貼るだけ
- OGP/oEmbedで自動プレビュー生成
- タグ選択（ジャンル、雰囲気、テーマ）

### Step 2: 「何を作りたいですか？」

```
投稿完了 → プロジェクトタイプ選択
```

| プロジェクトタイプ | 説明 | 必要分野 |
|-------------------|------|----------|
| MV制作 | 音楽+イラスト+映像 | イラスト、映像編集 |
| アルバムアート | 音楽+イラスト | イラスト |
| コラボ曲 | 作曲+作曲 | 作曲 |
| アニメーション | イラスト+映像 | 映像編集 |
| 自由コラボ | 制限なし | 全分野 |

### Step 3: レコメンド

```
タグベースで必要分野の作品を推薦
```

- ユーザーの投稿タグと類似するタグを持つ他分野の作品を表示
- 例: 「ローファイ」「夜」タグの音楽 → 「ローファイ」「夜景」タグのイラストを推薦

## 6. MVPスコープ

### 含む (MVP)

| 機能 | 説明 | 優先度 |
|------|------|--------|
| 外部リンクベースの作品投稿 | URL貼り付け → oEmbed/OGPプレビュー → タグ選択 | P0 |
| クリエイタープロフィール | 分野、自己紹介、投稿作品一覧 | P0 |
| レコメンド/作品マッチング | タグベースの作品推薦 | P0 |
| 探索 | タグ・分野フィルター付き作品一覧 | P0 |

### 含まない (Post-MVP)

| 機能 | 理由 |
|------|------|
| プロジェクトボード | MVP後の検証結果によって設計 |
| AIレコメンド | v1はタグマッチングで十分 |
| 決済/コミッション | 無料コラボに集中 |
| メディアストレージ | 外部リンクで代替 |
| DM/チャット | 初期はSNSリンクで代替 |

## 7. レコメンドロードマップ

| バージョン | 方式 | 説明 |
|-----------|------|------|
| **v1** | タグマッチング | 同じタグを持つ異分野の作品を推薦。シンプルだが効果的 |
| **v2** | コラボ履歴 + 人気度ウェイト | コラボ成立履歴と作品の人気度を加味したスコアリング |
| **v3** | エンベディングベース cross-domain類似度 | テキスト/画像/音声エンベディングによるクロスドメイン類似度計算 |

### v1 タグマッチングの詳細

```
ユーザーAの投稿: 音楽 [ローファイ, 夜, チル]
                    ↓ タグ一致検索
推薦結果: イラスト [ローファイ, 夜景, 静か] → 一致度: 2/3
         映像    [チル, 夜, アニメーション] → 一致度: 2/3
```

## 8. コールドスタート戦略

1. **シードコンテンツ投入** — 運営者自身が20〜30件の作品を投稿し、各分野・タグを網羅
2. **Twitter/Xクリエイターコミュニティへの告知** — #作曲 #イラスト #映像制作 タグでの宣伝
3. **piapro既存ユーザーへのリーチ** — ボカロP・絵師コミュニティへの直接アプローチ
4. **コラボ成功事例の公開** — 最初のコラボMVを制作し、プラットフォームの価値を実証
5. **招待制ベータ** — 初期は招待制で質の高いクリエイターを集め、コミュニティ文化を形成

---

## 9. 主要リスク

| リスク | 影響度 | 発生確率 | 緩和策 |
|--------|--------|----------|--------|
| コールドスタート問題（初期コンテンツ不足） | 高 | 高 | シードコンテンツ + 招待制ベータで段階的に成長 |
| タグマッチングの精度不足 | 中 | 中 | タグプリセットで品質担保 + v2以降でアルゴリズム改善 |
| ライセンストラブル | 高 | 低 | デフォルトライセンスの明示 + 投稿時の同意フロー |
| クリエイターの離脱（コラボ不成立） | 高 | 中 | レコメンド精度向上 + コラボ成功事例の可視化 |

---

## まとめ

Fugueは**「無料 + 作品ベースのコラボマッチング」**という空白地帯を狙うプラットフォームである。

ヘッドフォンをつけて絵筆を持ったフグ（Fugue/フグの語呂合わせ）をマスコットに、趣味クリエイターが気軽に作品を共有し、作品同士の相性でコラボ相手を見つけられる世界を目指す。

MVPでは外部リンクベースの作品投稿とタグマッチングレコメンドに集中し、小さく始めて検証する。
